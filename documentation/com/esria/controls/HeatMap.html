colorFunctionproperty
colorFunction:Function  [read-write]

A function used to determine the color of a region. The default colorFunction uses the fillColors and fillRatios styles to calculate the colors.

The color function has the following signature:

function( value:Number ):uint

Where the value argument is on a scale between zero and one that represents a comparison between the different region weight values.

horizontalFieldproperty 
horizontalField:String  [read-write]

The field used to determine the horizontal position an item. The horizontal position is the absolute position of the item from the left edge of the component, in pixels.

If the horizontalFunction is set, the horizontalField is ignored.

This property can be used as the source for data binding.

horizontalFunctionproperty 
horizontalFunction:Function  [read-write]

A function used to determine the horizontal position an item. The horizontal position is the absolute position of the item from the left edge of the component, in pixels.

The horizontalFunction takes precedence over the horizontalField when determining the horizontal position.

The horizontalFunction has the following signature:

function( item:Object ):Number

Where the item argument is an item from the heatmap's data provider, and the return value is the horizontal position of the item, in pixels.

This property can be used as the source for data binding.

weightFieldproperty 
weightField:String  [read-write]

The field used to determine the weight of an item. The weight of items is used to calculate the color of a heatmap region in which an item appears. By default, the weight of every item in the heatmap is 1. The weightField property may be used to give items differing weight values.

This property can be used as the source for data binding.

weightFunctionproperty 
weightFunction:Function  [read-write]

A function used to determine the weight of an item. The weight of items is used to calculate the color of a heatmap region in which an item appears. By default, the weight of every item in the heatmap is 1. The weightField property may be used to give items differing weight values.

The weightFunction has the following signature:

function( item:Object ):Number

Where the item argument is an item from the heatmap's data provider, and the return value is a Number representing the "weight" or value of an item used to determine the color of a region.

This property can be used as the source for data binding.
